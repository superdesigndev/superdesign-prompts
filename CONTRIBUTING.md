# Contributing

This is a read-only mirror of the [Superdesign prompt library](https://superdesign.dev/library), auto-synced and ranked by real usage.

## Add a prompt
Create it in the app at [superdesign.dev](https://superdesign.dev). The most-used prompts sync into this repo automatically.

## Fix this repo
Typos, broken links, or categorization fixes: open a PR. Please keep one change per PR and reference the prompt slug.
