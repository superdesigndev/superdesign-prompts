# Contributing

Read-only mirror of the [Superdesign prompt library](https://superdesign.dev/library), auto-synced and ranked by real usage.

## Add a prompt
Create it in the app at [superdesign.dev](https://superdesign.dev); top prompts sync here automatically.

## Fix this repo
Typos/links/categorization: open a PR, one change per PR, reference the slug.
