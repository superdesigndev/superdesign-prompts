## 在你的编程 Agent 里使用这些提示词

三种方式，任选其一：

**1 · 网页一键生成** —— 打开任意提示词，点击 **▶ Try live**，在 Superdesign 画布上直接生成，零配置。

**2 · 在 Claude Code / Cursor 等 Agent 内使用** —— 安装一次 [Superdesign skill](https://github.com/superdesigndev/superdesign-skill)，即可按 slug 拉取任意提示词：

```bash
# 适用于 Claude Code、Cursor 及任何支持 Claude Code skill 的 Agent
superdesign get-prompts --slugs "analytics-dashboard,brutalist-e-commerce-page" --json
```

**3 · 直接复制原始提示词** —— 每个提示词目录下都有 `README.md` 附完整提示词，或从 [`prompts.json`](prompts.json) / [`prompts.csv`](prompts.csv) 批量获取。
